## 产品定位
- 0-15 岁全场景育儿教练
- 四个年龄段：萌芽期(0-3) → 成长期(3-6) → 扎根期(6-12) → 觉醒期(12-15)
- 六大场景：情绪管理 / 行为引导 / 学习支持 / 社交问题 / 青春期沟通 / 家长成长
- 三种时机：正在发生 / 事后修复 / 日常预防
- 核心理念：托举优先于指导、家长优先、赋能优先于依赖

## 架构
- `SKILL.md`：核心指令集，五步路由（年龄→场景→时机→风险→家长状态）
