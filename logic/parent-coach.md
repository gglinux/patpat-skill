# 家长托举指南 (Parent Coach Guide)

本指南为 Agent 提供针对家长的即时支持（Scaffolding）策略，旨在帮助家长在孩子情绪风暴时保持冷静并提供正确引导。

## 1. 共情话术 (Empathy Phrases)
*   **原则**：命名情绪，不做评判。
*   **示例**：
    *   “我看到你现在很难受，是因为……”
    *   “如果你觉得这件事不公平，爸爸/妈妈完全理解。”
    *   “这个小火苗现在让你感觉很烫对吗？”

## 2. 边界话术 (Boundary Setting)
*   **原则**：温和地限制行为，而非限制情绪。
*   **示例**：
    *   “你可以生气，但我们不能踢门。”
    *   “我不允许你伤害自己，我们要把手停下来。”
    *   “等你不那么难受了，我们再一起解决这个问题。”

## 3. 即时动作 (Actionable Advice)
*   **原则**：低门槛、身体接触、物理调节。
*   **示例**：
    *   **10秒拥抱**：深长且稳定的拥抱，帮助孩子调节神经系统。
    *   **吹气球**：引导孩子像吹气球一样深呼吸。
    *   **感官描述**：让孩子寻找房间里 3 个红色的东西，转移注意力。

## 4. 事后修复 (Repair Script)
*   **原则**：关系连接，而非再次训诫。
*   **建议脚本**：
    *   “刚才我们都觉得很难受，但我很高兴后来你冷静下来了。”
    *   “当你感觉好一点的时候，你想告诉我刚才发生了什么吗？”
    *   “不管刚才发生了什么，我都依然爱你。”
