# 科学育儿（摸摸头）| AI 情绪共调节技能

**科学育儿（摸摸头）** 是一款服务 3-8 岁儿童及家长的 AI 情绪共调节技能。它覆盖情绪爆发当下、事后修复、以及平时练习三个场景，目标是先托住家长，再托住孩子，帮助家庭形成可重复使用的情绪支持仪式。

---

## ✨ 核心特性

- **三大入口模式**：`正在爆发` / `事后修复` / `平时练习`
- **低负担分流**：仅在必要时询问年龄段、可见状态和家长状态
- **家长优先托举**：固定的「此刻先做 / 现在可以说 / 现在先别做」脚手架结构
- **可重复家庭仪式**：情绪小生物 → 身体信号 → 一起做动作 → 再选下一步
- **场景优先指导**：从常见育儿场景出发，映射到情绪机制，不做道德说教
- **风险分级边界**：绿色 / 黄色 / 红色三级路由，确保安全

---

## 🛠️ 仓库结构

| 文件 | 说明 |
|------|------|
| `VISION.md` | 产品愿景、核心理念、痛点分析与体验设计 |
| `SKILL.md` | 核心指令集，定义 AI 的行为逻辑和路由规则 |
| `logic/story-engine.md` | 模式专属叙事规则、年龄分层、仪式结构与输出契约 |
| `logic/emotion-creatures.md` | 情绪小生物↔身体信号映射表与场景指导 |
| `logic/parent-coach.md` | 家长支持的权威指南：分流、自救、修复脚本、共调节动作库 |

---

## 🤖 如何使用

1. 将 `SKILL.md` 注入 AI 智能体的系统提示词中
2. `logic/` 目录下的文件作为运行时规则集
3. 每个请求先路由到三大入口模式之一，再生成内容
4. 回复结构保持：家长指导 → 孩子内容 → 共同动作 → 后续提醒
5. 如前端需要结构化渲染，将相同段落映射为 JSON 格式

## 🎯 产品初衷

「科学育儿（摸摸头）」不是来替代家长或评判家长的。
它是一个轻量级的情绪脚手架工具，帮助家庭：
- 以更少的羞耻感度过艰难时刻
- 在冲突之后修复亲子连接
- 在下一场风暴来临之前练习情绪技能
